# 🔐 Password Generator (Python)

This is a simple Python project that generates strong and secure passwords using random characters.

## 🚀 Features
- Generate secure passwords
- Custom password length
- Uses letters, numbers, and special characters

## 🛠️ Technologies Used
- Python
- random module
- string module

## ▶️ How to Run
1. Install Python (if not installed)
2. Run the file:
   python password_generator.py

## 💡 Example
Enter password length: 10  
Output: @G7#kP2!xQ
